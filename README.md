# VEGA SoC

**VEGA SoC** is a multi-power-domain RTL-to-GDSII ASIC implementation project featuring:

- **Low Power** - Optimized for minimal power consumption
- **Power Management** - Comprehensive power management capabilities
- **Retention Logic** - Advanced retention mechanisms for power-gated domains
- **IEEE 1801 UPF** - Full compliance with UPF (Unified Power Format) standards
- **RTL-to-GDSII Flow** - Complete design flow from RTL to final GDSII layout

## Project Overview

This repository contains the complete implementation of the VEGA System-on-Chip (SoC), a sophisticated multi-power-domain ASIC designed for advanced low-power applications.

### Key Features

- Multi-power-domain architecture
- Unified Power Format (UPF) compliance
- Retention logic for power-gated regions
- Complete RTL-to-GDSII design flow
- Optimized for low-power operation
